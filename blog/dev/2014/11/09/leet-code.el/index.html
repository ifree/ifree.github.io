leetcode.el

Nov 09, 2014

以前我写过一个 meacs 的 oj 工具 今天又写了一个 leet-code.el, 至于为什么不继续扩 展以前的而重新写一个呢. 主要原因是我现在 elisp 的功力有限, 没法把代码写的很漂亮. 以后有机会了可能会把他们合并了, 但是, 现在就先这样吧.

关于 leetcode, 我断断续续刷过一段时间, 里面的题也比较经典, 也有比较难的. 准备继 续刷了, 每天再挤一个小时出来 orz… 关于这个工具, 主要是让你能更方便专注的刷题, 也节省浏览器和编辑器之间的切换.

主要功能

  • 登录
  • 提交代码(文件, region)
  • 查看提交结果
  • 获取题目说明
  • 获取默认代码

    有什么问题就直接提 issue 吧, 希望这个工具能帮你节约一点时间.
